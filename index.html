TabTracker

Open-source MIT Licensed.

Tab Summaries 🛡️

Clear overviews of all open tabs and their content summaries enabling efficient navigation between different tabs.

Active/Inactive Status 🚀

Easily see which tabs are active or inactive, enabling easy tab decluttering

Tab Hyperlinking📈

TabTracker can replace Chrome tab displays with hyperlinking that takes you directly to you desired tab by clicking on the tab name

Easy Decluttering🦉

Simple user interface that lets users delete all inactive tabs with just one button.

Tab Search✨

Users can search for any tab in their browser through the extension, reducing the time taken find a desired tab.

All Browser View🧰

Tabs are separated by browser placement, providing a holistic overview of all open Chome tabs.
